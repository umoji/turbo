# openni-lesson

演習用のOpenNIサンプルコード

## Requirements

* OpenNI v1.5.2.23
* NITE v1.5.2.21
* OpenGL

## Usage

ビルドは各サンプルのディレクトリに入って行う。

    $ cd 01-depth-test
    $ make

`bin/[arch]-Release`ディレクトリに実行ファイルが生成される。

    $ cd bin/[arch]-Release
    $ ./01-depth-test
